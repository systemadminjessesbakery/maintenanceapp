Stores Master

The Stores Master section allows you to manage all store-related information and delivery schedules.

Adding a New Store

  1. Click the "Add New Store" button in the top menu
  2. Fill in the required fields:
    • Store ID (unique identifier)
    • Store Name
    • Address
    • Region
    • State
  3. Set delivery days by checking the appropriate boxes (Sunday through Saturday)
  4. Set special delivery days if applicable
  5. Click "Save" to create the store

Editing Store Information

You can edit any store's information by:

  1. Clicking on any field in the store's row
  2. Making the necessary changes
  3. Clicking outside the field to save changes
Tip: Use the search and filter options to quickly find specific stores.

Products Master

The Products Master section manages your product catalog, including pricing and specifications.

Adding a New Product

  1. Click the "Add New Product" button
  2. Fill in the required fields:
    • Product ID (unique identifier)
    • Product Description
    • Product Family (category)
  3. Add retailer-specific codes (Woolworths, Coles, etc.)
  4. Set baking specifications:
    • Baking UOM (Unit of Measure)
    • Baking Quantity
    • Unit Per Product
  5. Enter pricing information:
    • Wholesale Cost (AUD)
    • RRP (AUD)
  6. Add Production Description if different from regular description
  7. Click "Save" to create the product
Note: All monetary values should be entered in Australian Dollars (AUD).

Standing Orders

Manage regular delivery orders for each store-product combination.

Creating a Standing Order

  1. Click "Add New Standing Order"
  2. Select the store from the dropdown
  3. Select the product from the dropdown
  4. Enter quantities for each day of the week
  5. The week total will be calculated automatically
  6. Click "Save" to create the standing order

Modifying Standing Orders

You can modify existing standing orders by:

  1. Clicking on any quantity field
  2. Entering the new quantity
  3. Changes are saved automatically
Warning: Changes to standing orders will affect future delivery schedules.

Adjustments

Make temporary modifications to delivery quantities for specific stores and products.

Creating an Adjustment

  1. Click "Add New Adjustment"
  2. Select the store and product
  3. Enter adjustment quantities for each day
  4. The week total will be calculated automatically
  5. Click "Save" to create the adjustment
Tip: Use adjustments for temporary changes, while standing orders are for regular patterns.

Final Production

View and manage the final production numbers for each product.

Viewing Production Numbers

  • Use the filters to view specific stores or products
  • Daily quantities are displayed in the table
  • Week totals are calculated automatically

Exporting Production Data

  1. Click "Export to Excel" for spreadsheet format
  2. Click "Export to PDF" for printable format

Final Delivery

View and manage the final delivery schedules for each store.

Viewing Delivery Schedules

  • Use the filters to view specific stores or products
  • Daily delivery quantities are displayed in the table
  • Week totals are calculated automatically

Exporting Delivery Data

  1. Click "Export to Excel" for spreadsheet format
  2. Click "Export to PDF" for printable format
Note: Final delivery numbers are calculated based on standing orders and adjustments.
